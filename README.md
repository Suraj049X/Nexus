# NEXUS 🚀

**NEXUS** is my personal digital playground for building, experimenting, and learning web development. It represents the intersection of code, creativity, and technology, where every project contributes to my journey as a developer.

## 🌐 Live Demo

🔗 https://suraj049x.github.io/Nexus/

## ✨ Features

* Modern dark-themed portfolio design
* Smooth scrolling navigation
* Responsive layout for desktop and mobile devices
* Auto-scrolling experiment sections
* Custom styled scrollbars
* Interactive project showcase
* Contact section with social links

* Text Level Tags
* Border Properties
* Special Characters

### 2. Lists & Navigation

* Different Types of Lists
* Nested Lists
* Internal Links
* External Links
* Images & Alignment
* Link Colors

### 3. Images & Styling

* Image Attributes
* Hyperlinked Images
* Background Properties
* CSS Stylesheets
* Responsive Web Design

### 4. Tables, Layouts & Forms

* Table Properties
* Nested Tables
* Rowspan & Colspan
* Frame Properties
* Form Properties
