HTML (HyperText Markup Language) is the foundational language for web development.
HTML uses elements like tags to define headings, paragraphs, and links.





Each HTML document begins with a <!DOCTYPE html> declaration to define its structure.
 Tags are enclosed within angle brackets, such as <p>, <h1>, and <a>.





HTML documents are structured using tags like , <html>, <head>, and <body>
Example of Paragraph Tag: 

<p>This is a simple HTML paragraph.<p>
